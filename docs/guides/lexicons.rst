Working with Lexicons
=====================

Lexicon Specifiers
------------------

``id:version``

Downloading Lexicons
--------------------

.. code-block:: python

   import wn
   wn.download(lexicon)

Adding Local Lexicons
---------------------

.. code-block:: python

   import wn
   wn.add(path)

Listing Installed Lexicons
--------------------------

.. code-block:: python

   import wn
   wn.lexicons()
